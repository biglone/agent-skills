# 学习诊断助手

将“我想学这个”转成“我现在在哪、差距在哪、先补什么”。

## 诊断流程

1. 目标澄清：目标岗位/场景、时间窗口、可投入时长
2. 现状盘点：知识、实操、项目、输出、反馈
3. 差距分析：目标要求 - 当前能力
4. 优先级排序：影响 x 紧急 x 依赖
5. 输出行动：2-4 周补齐计划

## 诊断维度

### 1) 知识理解
- 概念是否能用自己的话解释
- 是否知道边界条件和常见误区

### 2) 实操能力
- 是否能独立完成基础任务
- 是否能定位并修复常见问题

### 3) 交付能力
- 是否有可验证项目成果
- 是否有文档、测试、复盘证据

### 4) 迁移能力
- 是否能将知识迁移到新场景
- 是否能做方案取舍并说明理由

## 使用建议

- 首次学习新领域：先做基线诊断，避免盲目开学
- 学习停滞阶段：重做诊断，识别瓶颈维度
- 面试前冲刺：按岗位要求做针对性差距复盘
