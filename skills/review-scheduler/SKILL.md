# 复习调度助手

将“知道要复习”转成“何时复习、复习什么、何时调整计划”。

## 调度流程

1. 盘点学习单元：概念、题集、项目
2. 设定复习窗口：日/周/月
3. 生成复习日历：间隔重复 + 优先级
4. 周度回顾：完成率、遗忘率、难点分布
5. 动态重排：删减/延期/强化

## 间隔重复策略

```text
Day 0  首次学习
Day 1  快速复习
Day 3  第二次复习
Day 7  第三次复习
Day 14 第四次复习
Day 30 第五次复习
```

## 重规划触发条件

- 连续 2 周完成率 < 70%
- 同一主题复测正确率 < 60%
- 学习负荷明显超出可用时间

## 使用建议

- 每周固定一个“重排窗口”（如周日晚上）
- 复习内容优先覆盖高价值和高遗忘主题
- 与测验结果联动，错题优先进入下一轮复习
