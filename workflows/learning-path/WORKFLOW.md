# 学习路径工作流

## 工作流概述

```
基线诊断 → 设定目标 → 规划路径 → 执行学习 → 阶段测验 → 整理笔记 → 知识内化 → 周度重规划 → 输出分享
```

## 流程步骤

### 步骤 1：基线诊断
**使用 Skill**：`learning-diagnostics`

学习开始前先确认：
- 当前能力水平
- 与目标的关键差距
- 优先补齐项（P0/P1/P2）

**触发方式**：
```
请帮我做 [主题] 的学习诊断，并给出优先级差距清单
```

**输出**：
- 诊断评分表
- 差距优先级列表
- 2-4 周补齐建议

### 步骤 2：设定学习目标
**使用 Skill**：`goal-setter`

明确学习目标：
- 要学什么
- 达到什么水平
- 什么时间完成

**触发方式**：
```
我想学习 [技能/知识]，请帮我设定学习目标
```

**输出**：
- SMART 目标描述
- 分阶段里程碑
- 成功标准定义

### 步骤 3：规划学习路径
**使用 Skill**：`learning-tracker`

制定学习计划：
- 学习内容分解
- 资源整理
- 时间安排

**触发方式**：
```
请帮我制定学习 [技能] 的详细计划
```

**输出**：
- 分阶段学习计划
- 推荐学习资源
- 每周学习安排

### 步骤 4：执行学习
**使用 Skill**：`concept-explainer`

学习过程中：
- 理解新概念
- 解决疑问
- 深化理解

**触发方式**：
```
请解释一下 [概念] 是什么？
```

**输出**：
- 概念的清晰解释
- 简单易懂的类比
- 实际应用示例

### 步骤 5：阶段测验
**使用 Skill**：`quiz-generator`

每完成一个学习阶段就做验收：
- 概念掌握度检查
- 题目/实操表现评估
- 错题与薄弱点识别

**触发方式**：
```
请基于 [主题/章节] 生成一套测验，并附评分标准
```

**输出**：
- 分层题集（基础/进阶/实战）
- 参考答案与评分点
- 错题复盘表

### 步骤 6：整理笔记
**使用 Skill**：`note-taker`

学习后整理：
- 提取关键点
- 结构化笔记
- 建立联系

**触发方式**：
```
请帮我整理这段学习内容的笔记
```

**输出**：
- 结构化笔记
- 关键概念总结
- 复习要点

### 步骤 7：知识内化
**使用 Skill**：`knowledge-base`

构建知识体系：
- 整合新旧知识
- 建立知识地图
- 定期复习

**触发方式**：
```
请帮我将 [主题] 的学习内容整合到知识库
```

**输出**：
- 知识卡片
- 知识关联图
- 复习计划

### 步骤 8：周度重规划
**使用 Skill**：`review-scheduler`

根据一周执行结果调整节奏：
- 完成率与遗忘率统计
- 保留/延后/加强项目重排
- 复习日历更新

**触发方式**：
```
请根据我本周学习数据，重排下周计划和复习安排
```

**输出**：
- 周度回顾结果
- 下周学习与复习日历
- 调整后的优先级

### 步骤 9：输出分享
**使用 Skill**：`tutorial-creator`

通过教授巩固：
- 写教程/博客
- 制作分享材料
- 费曼学习法实践

**触发方式**：
```
请帮我将 [主题] 的学习成果整理成一篇教程
```

**输出**：
- 完整教程文档
- 循序渐进的结构
- 练习和检验点

## 快速启动

### 启动学习
```
我想学习 [技能/知识]：
- 当前水平：[描述]
- 目标水平：[描述]
- 可用时间：每周 X 小时
请帮我规划学习路径
```

### 学习中求助
```
我在学习 [主题] 时遇到了问题：
[问题描述]
请帮我解释
```

### 学习总结
```
我完成了 [阶段/课程] 的学习，请帮我：
1. 整理学习笔记
2. 生成阶段测验并评估掌握度
3. 制定下周复习和重规划方案
```

### 输出巩固
```
我想把 [主题] 的学习成果写成教程分享：
- 目标读者：[描述]
- 重点内容：[列表]
```

## 学习检查清单

- [ ] 明确学习目标
- [ ] 评估当前水平
- [ ] 制定学习计划
- [ ] 准备学习资源

### 学习中
- [ ] 每日/周记录进度
- [ ] 阶段学习后做测验
- [ ] 及时整理笔记
- [ ] 解决遇到的问题
- [ ] 完成练习和实践

### 学习后
- [ ] 总结学习收获
- [ ] 建立知识关联
- [ ] 制定复习计划
- [ ] 每周重规划一次
- [ ] 输出分享巩固

## 学习效率技巧

### 番茄工作法
```
25分钟专注学习 → 5分钟休息 → 重复
每4个番茄 → 长休息15-30分钟
```

### 间隔重复
```
Day 1: 学习
Day 2: 第一次复习
Day 4: 第二次复习
Day 7: 第三次复习
Day 14: 第四次复习
```

### 费曼学习法
```
1. 学习概念
2. 教授给别人（或假装教授）
3. 发现理解不清的地方
4. 回去深入学习
5. 用更简单的语言重新解释
```
